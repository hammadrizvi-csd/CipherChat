# 🛡️ CipherChat - Next-Gen Secure Messaging

CipherChat ek end-to-end encrypted real-time chat platform hai jo privacy-first approach par banaya gaya hai. Isme humne modern tech stack ka use karke ek smooth aur secure messaging experience diya hai.

## ✨ Key Highlights
* **🔒 End-to-End Encryption**: CryptoJS (AES) ka use karke messages ko browser side par hi encrypt kiya jata hai.
* **⚡ Real-Time Engine**: Socket.io se instantaneous message delivery aur typing indicators milte hain.
* **📩 Advanced Ticks**: WhatsApp-style delivery status (Sent, Delivered, aur Read Blue Ticks).
* **🌓 Dual Theme**: Modern Light aur Dark mode support.
* **🖼️ Media Handling**: In-app image cropping aur secure file storage system.

## 🛠️ Built With
* **Backend**: Node.js & Express.js
* **Database**: MongoDB Atlas with Mongoose
* **Real-time**: Socket.io
* **Security**: CryptoJS, JWT, & Bcrypt
